# InventoryHub

InventoryHub es una aplicación full stack desarrollada con Blazor WebAssembly (frontend) y Minimal API (backend).

## Funcionalidades

* Visualización de productos desde una API
* Comunicación entre frontend y backend
* Manejo de JSON estructurado
* Resolución de errores de integración
* Optimización del rendimiento

## Tecnologías

* Blazor WebAssembly
* .NET Minimal API
* C#
* JSON

## Ejecución

1. Ejecutar backend:

```
cd ServerApp
dotnet run
```

2. Ejecutar frontend:

```
cd ClientApp
dotnet run
```

3. Abrir en navegador:

```
/fetchproducts
```
